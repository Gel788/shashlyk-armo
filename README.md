# 🔥 Шашлык Армо

Сайт настоящего армянского шашлычного ресторана с доставкой по звонку.

## Технологии

- Чистый HTML5 / CSS3 / Vanilla JS
- Google Fonts: Playfair Display, Inter, Oswald
- Анимации на CSS (угольки, огонь, пульсации)
- Адаптивный дизайн (mobile-first)

## Разделы

- **Hero** — полноэкранный с летящими искрами и огненным заголовком
- **О нас** — история и преимущества
- **Меню** — 4 категории, карточки блюд с 3D-hover эффектом
- **Доставка** — анимированный мокап телефона, шаги заказа
- **Отзывы** — рейтинги с платформ (Google, Яндекс, 2ГИС)
- **Контакты** — стилизованная карта, соцсети

## Особенности дизайна

- Прелоадер с анимацией огня
- Промо-баннер с бегущей строкой
- Разделители секций в виде живого огня
- Плавающая кнопка звонка с пульсирующим кольцом
- Счётчики статистики с анимацией
- Параллакс Hero при скролле
- Полная адаптивность под мобильные

## Запуск

Просто открой `index.html` в браузере.
